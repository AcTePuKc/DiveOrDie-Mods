# DIVE or DIE Mods

Open-source BepInEx 5 mods for the Windows x64 Mono build of **DIVE or DIE – Children of Rain**.

## Projects

- [Skip Intro](Skip_intro/) — stable standalone preloader patcher with English and Bulgarian instructions. Binary releases are available from this repository's Releases page.
- [Bulgarian Language](Bulgarian_language/) — Bulgarian localization source and translation table. The language selector is still experimental, so no binary GitHub Release is published yet.

The projects are independent. Players may install either mod by itself or both together.

This repository does not contain game assemblies, extracted game assets, BepInEx binaries, logs, decompiled files, or private development notes.

# Модове за DIVE or DIE

Модове с отворен код за BepInEx 5 и Windows x64 Mono версията на **DIVE or DIE – Children of Rain**.

## Проекти

- [Skip Intro](Skip_intro/) — стабилен самостоятелен мод за прескачане на началните екрани с упътване на английски и български. Готовият архив е в раздела Releases.
- [Bulgarian Language](Bulgarian_language/) — изходен код и преводаческа таблица за българската локализация. Езиковото меню още е експериментално, затова засега няма готово двоично издание в GitHub.

Двата проекта са независими. Може да се използва само единият или и двата едновременно.

Хранилището не съдържа библиотеки от играта, извлечени ресурси, файлове на BepInEx, дневници, декомпилирани файлове или лични работни бележки.
